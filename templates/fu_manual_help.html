手動符入力ヘルプ

手動符入力とは：自動計算ではなく、自分で符を計算して入力する機能です。

手動符入力の使い方

「手動符入力」にチェックを入れると、自動符計算が無効になり、自分で符を入力できるようになります。

例：手動符入力にチェック → 符の欄に「40」と入力

手動符入力が必要な場合

  • 複雑な手牌で自動計算が難しい場合
  • 特殊な和了形の場合
  • 符計算の練習をしたい場合
  • 自動計算に不具合がある場合

符の計算方法

符は以下の要素から計算されます：

要素 符 説明
基本符 20符 すべての和了に付く
和了方法 ツモ+2符
ロン+10符		 ツモ和了かロン和了か
面子符 刻子・槓子による符 暗刻・明刻・暗槓・明槓
雀頭符 役牌+2符 場風・自風・三元牌
待ち符 特殊待ち+2符 辺張・嵌張・単騎待ち

符計算の例

例1：平和・ツモ・両面待ち
基本符20符 + ツモ2符 + 両面待ち0符 = 22符 → 30符（切り上げ）
例2：ロン・単騎待ち・暗刻（老頭牌）
基本符20符 + ロン10符 + 単騎待ち2符 + 暗刻（老頭牌）8符 = 40符
例3：複雑な手牌
基本符20符 + ツモ2符 + 暗刻×2（16符） + 明刻×1（4符） + 役牌2符 + 単騎待ち2符 = 46符 → 50符（切り上げ）

注意事項

  • 符は10の倍数に切り上げます（例：34符 → 40符）
  • 平和は固定で30符になります
  • 七対子は固定で25符になります
  • 国士無双は固定で25符になります
  • 手動入力時は自動計算が無効になります

符計算のコツ

  1. 基本符20符から始める
  2. 和了方法の符を加える（ツモ+2符、ロン+10符）
  3. 刻子・槓子の符を計算する
  4. 役牌の雀頭があれば+2符
  5. 特殊待ちがあれば+2符
  6. 10の倍数に切り上げる
