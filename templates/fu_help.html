符（ふ）計算ヘルプ

符とは：麻雀の手牌の形に基づいて計算される基本点です。符は翻数と組み合わせて最終的な点数を決定します。

基本符（20符）

すべての和了に基本符として20符が与えられます。

面子符

刻子符

刻子の種類 中張牌（2-8） 老頭牌（1、9）・字牌
暗刻 8符 32符
明刻 4符 16符
暗槓 32符 64符
明槓 16符 32符
例：暗刻「5-5-5」= 8符、明刻「東-東-東」= 16符

順子符

順子には符は付きません（0符）。

例：順子「1-2-3」= 0符

雀頭符

役牌（場風、自風、三元牌）の対子には2符が付きます。

例：場風が東の場合、「東-東」= 2符

待ち符

待ちの種類 符
両面待ち 0符
辺張待ち 2符
嵌張待ち 2符
単騎待ち 2符
例：「1-2」待ち（3待ち）= 辺張待ちで2符

符計算の例

例1：
暗刻「5-5-5」（8符）+ 明刻「東-東-東」（16符）+ 順子「1-2-3」（0符）+ 順子「7-8-9」（0符）+ 雀頭「2-2」（0符）+ 両面待ち（0符）
= 20 + 8 + 16 + 0 + 0 + 0 + 0 = 44符 → 50符（切り上げ）
例2：
暗刻「1-1-1」（32符）+ 暗刻「9-9-9」（32符）+ 明刻「白-白-白」（16符）+ 順子「2-3-4」（0符）+ 雀頭「東-東」（2符）+ 単騎待ち（2符）
= 20 + 32 + 32 + 16 + 0 + 2 + 2 = 104符 → 110符（切り上げ）

注意事項

  • 符は10の倍数に切り上げます（例：34符 → 40符）
  • 平和は固定で30符になります
  • 七対子は固定で25符になります
  • 国士無双は固定で25符になります
